##### 1. DESKTOP:-

This is a type of personal computer designed to fit on or under a desk. It typically consists of a separate monitor, computer case (which houses the CPU, motherboard, and other components), keyboard, and mouse. Desktops are usually more powerful and customizable than laptops, and they're commonly used in office environments and for tasks that require significant computing power
##### 2. STORAGE:-

 Storage is a fundamental aspect of computing and refers to the method of saving and retrieving data on a computer or other electronic device. There are various types of storage, each with its own characteristics, uses, and advantages.
 1. Primary Storage (Volatile)
 2. econdary Storage (Non-Volatile)
 3. Network Storage: This refers to storage that is accessible over a network
 4. Cloud Storage: Cloud storage involves storing data on remote servers accessed via the internet.

##### 3. NETWORK:-

A network refers to a collection of interconnected devices that can communicate with each other and share resources. Networks are essential for modern computing, enabling everything from internet access to file sharing and communication between devices.
1. Local Area Network (LAN): A LAN is a network that connects devices within a limited area, such as a home, office, or campus. It allows for high-speed data transfer and resource sharing among connected devices.
2. Wide Area Network (WAN):A WAN covers a broad geographic area, often spanning cities, countries, or even continents. It connects multiple LANs and enables communication over long distances
3. Metropolitan Area Network (MAN):A MAN covers a city or a large campus. It is larger than a LAN but smaller than a WAN.
4. Personal Area Network (PAN):A PAN is a small network used for connecting devices within the range of an individual person, typically within a few meters.

##### 4. OPERATING SYSTEM:-

An operating system (OS) is essential software that manages computer hardware and software resources and provides common services for computer programs. It acts as an intermediary between users and the computer hardware, enabling user-friendly interaction and ensuring efficient operation. 
Some commonly used operating systems are:-
1. Microsoft Windows
2. macOS
3. Linux

 ###### 5. COMPUTATIION:-

 Computation in computer science refers to the process of performing calculations or processing data according to a set of instructions or algorithms. It encompasses a wide range of activities, from basic arithmetic operations to complex simulations and data analysis

##### 6. CPU:-

The Central Processing Unit (CPU) is often referred to as the "brain" of a computer. It performs the majority of the processing tasks and executes instructions from computer programs.  The CPU executes instructions from programs, which can be arithmetic operations, data manipulation, or control commands.

##### 7. GPU:-

The Graphics Processing Unit (GPU) is a specialized processor designed to handle the complex calculations required for rendering graphics and performing parallel processing tasks. While originally developed to accelerate rendering of images and video, GPUs are now also widely used for a variety of computational tasks beyond graphics. ##### 8.NIC:-

A Network Interface Card (NIC) is a crucial hardware component that enables a computer or other networked device to connect to a network. It facilitates communication between devices on a network, allowing them to send and receive data.

##### 9. CLOUD COMPUTING:-

Cloud computing is a transformative technology that enables on-demand delivery of computing resources, such as servers, storage, databases, networking, software, and more, over the internet. Instead of owning and maintaining physical hardware, users can access these resources through cloud service providers on a pay-as-you-go basis.

#### 10. BARE METAL SYSTEM:-

A bare metal system refers to a computing environment where software directly interacts with the hardware without the abstraction layer of a traditional operating system or virtualization. This term is often used in the context of servers and computing infrastructure. #### 11. ARM:-

RM (Advanced RISC Machine) is a family of computer processor architectures widely used in various computing devices, from smartphones and tablets to embedded systems and increasingly in laptops and servers. ARM processors are known for their efficiency, scalability, and versatility

#### 12. RISC:-

RISC (Reduced Instruction Set Computer) is a type of computer architecture that emphasizes a simplified instruction set and streamlined processing. The RISC design philosophy contrasts with CISC (Complex Instruction Set Computer) architectures, which have a more complex set of instructions. 

###### 13. CISC:-

CISC (Complex Instruction Set Computer) is a type of computer architecture that features a broad set of instructions with varying lengths and complexities. This design philosophy contrasts with RISC (Reduced Instruction Set Computer), which focuses on a smaller set of simple instructions.

##### 14. RISC V:-

ISC-V is an open-source, RISC (Reduced Instruction Set Computer) architecture designed to be a highly extensible and modular instruction set architecture (ISA). It represents a significant departure from traditional proprietary ISAs by offering a freely available and customizable framework that allows developers and researchers to create custom processors and systems

##### 15.. FPGA:-

FPGA (Field-Programmable Gate Array) is a type of integrated circuit that can be configured by the user after manufacturing. Unlike traditional fixed-function chips, FPGAs can be programmed to perform a wide range of tasks, making them highly versatile and adaptable for various applications.

#### 16. VIRTUAL MACHINE:-

A virtual machine (VM) is a software emulation of a physical computer that runs an operating system and applications just like a physical machine. VMs allow multiple operating systems to run on a single physical machine by creating isolated environments that mimic real hardware. A virtual machine is a software-based simulation of a physical computer. It uses virtualization technology to provide an environment that mimics a physical machine’s hardware and operating system.

##### 17. KERNEL:-

he kernel is a fundamental component of an operating system, acting as a bridge between the hardware and the user-level applications. It manages system resources, hardware interactions, and provides essential services needed for the operating system to function effectivey

###### 18. IP ADDRESS:-

An IP address (Internet Protocol address) is a unique numerical identifier assigned to each device connected to a network that uses the Internet Protocol for communication. It serves two primary purposes: identifying the host or network interface and providing the location of the host in the network.

#### 19. VPN:-

A VPN (Virtual Private Network) is a technology that creates a secure, encrypted connection over a less secure network, such as the internet. It provides privacy, security, and access to resources by connecting a user or device to a remote network. 
A VPN establishes a secure connection between a user’s device and a remote network through encryption, making data transmitted over the internet private and secure.

##### 20.HARDWARE:-

Hardware refers to the physical components of a computer system and related devices. It encompasses all the tangible parts that make up a computer and its peripherals, distinguishing them from software, which consists of programs and data

#### 21. FRONT-END:-

Front-end development refers to the part of web development that focuses on creating the user interface and user experience of a website or application. It involves everything that users interact with directly in their web browsers, including the layout, design, and functionality of a website or web application. 

##### 22.BACK-END :-

Back-end development refers to the server-side part of web development that focuses on managing and processing the data behind the scenes. It involves the creation and maintenance of the server, databases, and application logic that powers the front-end (user interface) of a website or web application. Back-end development ensures that data is stored, processed, and served to users efficiently and securely.

##### 23.ETHERNET :-

Ethernet is a widely used technology for local area networks (LANs) that provides a standard way to connect devices and enable communication within a network. It defines the physical and data link layer specifications for networking, allowing devices to communicate and share resources efficiently.

##### 24.MEMORY:- 

Memory in computing refers to the physical and logical devices that store data and instructions for use by the computer's processor. It plays a critical role in determining a computer's performance and capability to handle tasks.

#### 25. BLOCK CHAIN:-

Blockchain is a foundational technology that underpins various applications by offering a decentralized, secure, and transparent method for recording transactions and managing data. It involves multiple disciplines, including cryptography, distributed systems, and data structures. 

##### 26. DNS (Domain Name System):-

The Domain Name System (DNS) is a fundamental component of the internet's infrastructure, providing a method to translate human-friendly domain names into IP addresses that computers use to identify each other on the network. It plays a critical role in making the internet accessible and usable for everyday users.

##### 27. DPU (Data Processing Unit):-

A Data Processing Unit (DPU) is a specialized hardware component designed to handle data-centric tasks, such as data management, processing, and networking, with high efficiency. DPUs are increasingly relevant in modern computing architectures, particularly as data processing needs become more complex and data volumes continue to grow.

##### 28. API (Application Programming Interface):-

An Application Programming Interface (API) is a set of rules and protocols that allows different software applications to communicate with each other. APIs define the methods and data formats that applications use to request and exchange information. They are essential for enabling the integration and interaction between various software systems, services, and platforms.

#### 29. DSA (Data Structures and Algorithms):-

Data Structures and Algorithms (DSA) are fundamental concepts in computer science that involve organizing and manipulating data to efficiently solve problems. Mastery of DSA is crucial for software development, as it helps in optimizing performance, managing resources, and solving complex computational problems effectively.
